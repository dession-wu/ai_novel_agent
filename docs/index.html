快速开始

几分钟内启动并运行NOVA

1

克隆仓库

git clone https://github.com/dession-wu/ai-novel-agent.git
cd ai-novel-agent
2

安装依赖

python -m venv venv
venv\Scripts\activate  # Windows
# or
source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
3

启动服务

python main.py
# 访问 http://localhost:8000
